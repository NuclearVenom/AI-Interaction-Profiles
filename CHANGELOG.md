# Changelog

All notable documentation changes to this repository will be recorded here.

This project follows a simple version history rather than a software release process because it is currently a documentation-first proof-of-concept.

## [1.0.0] - 2026-06-28

### Added

- Fourteen copy-ready Interaction Profiles.
- Consolidated `profile.md` pages for each profile.
- Full and Concise profile versions.
- Profile-level `assets/.gitkeep` folders for future screenshots.
- `profiles/index.md`.
- `docs/comparison-matrix.md`.
- Mermaid diagrams for interaction flow and profile selection.

### Changed

- Updated README and supporting docs for Version 1.0.
- Improved profile navigation and discoverability.
- Standardized profile Quick Start language.
- Added one-line profile taglines.
- Reworked category README files as navigation pages.

### Not Included

- Application code.
- Browser extension design.
- API or software design.
- Formal evaluation results.

## [0.2.0] - 2026-06-27

### Changed

- Rewrote the README to emphasize the interaction concept, the problem, and the distinction from chatbots or task instruction libraries.
- Reframed the repository around Interaction Profiles, behavior profiles, and the interaction layer.
- Rewrote philosophy, problem, vision, FAQ, getting started, and profile-writing guidance for consistency.
- Expanded design principles with explanations, reasoning, and examples.
- Improved examples with concise comparisons for communication, brainstorming, planning, learning, clarification, and decision making.
- Replaced `profiles/lite`, `profiles/balanced`, and `profiles/deep` with `profiles/communication`, `profiles/thinking`, `profiles/daily`, and `profiles/integrated`.
- Reduced premature product language and clarified that the repository is not software.

### Added

- `docs/manifesto.md`.
- Profile category README files.
- Additional comparison patterns in the examples directory.

### Not Included

- Finished Interaction Profiles.
- Code.
- Browser extension design.
- API or software design.
- Formal evaluation results.

## [0.1.0] - 2026-06-27

### Added

- Initial repository documentation.
- Project README.
- Philosophy, problem, vision, FAQ, and getting started documents.
- Design principles for future Interaction Profiles.
- Placeholder profile directories for Lite, Balanced, and Deep profiles.
- Before-and-after examples and comparison gallery.
- Contribution, security, citation, license, and community files.

### Not Included

- Usable Interaction Profiles.
- Code.
- Browser extension design.
- API or software design.
- Formal evaluation results.
