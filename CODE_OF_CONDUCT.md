# Code Of Conduct

## Our Pledge

We want AI Interaction Profiles to be a thoughtful, respectful space for discussing human-AI interaction. Contributors are expected to communicate with care, assume good intent where reasonable, and prioritize clarity over status.

## Our Standards

Examples of behavior that supports this community include:

- giving constructive feedback;
- distinguishing critique of ideas from critique of people;
- welcoming beginners;
- explaining assumptions;
- avoiding condescension;
- respecting different communication styles and accessibility needs;
- acknowledging uncertainty.

Examples of unacceptable behavior include:

- harassment, intimidation, or personal attacks;
- discriminatory language or behavior;
- repeated dismissal of good-faith questions;
- publishing private information without permission;
- derailing discussions with unrelated promotion;
- using the project to shame people for how they communicate.

## Scope

This Code of Conduct applies in repository issues, pull requests, discussions, and any official project spaces.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or restrict participation when behavior conflicts with this Code of Conduct.

When possible, maintainers should explain moderation decisions clearly and proportionately.

## Reporting

If you experience or observe unacceptable behavior, please contact the maintainers through the appropriate repository channel.

## Guiding Principle

This project is about improving interaction without making people feel inadequate. The community should model that same principle.
