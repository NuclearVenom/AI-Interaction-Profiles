# Disclaimer

AI Interaction Profiles is an exploratory proof-of-concept. It is not a scientific study, clinical tool, educational certification, productivity guarantee, chatbot, or official integration with any AI provider.

The repository investigates whether Interaction Profiles may improve the quality of conversations with existing large language models. It does not claim that any profile will work for every user, model, task, or context.

## Not Software

This repository does not contain:

- application code;
- browser extension code;
- APIs;
- model wrappers;
- automation tools;
- technical build specifications.

Any future product ideas are research directions, not current functionality.

## No Model Modification

Interaction Profiles do not train, fine-tune, modify, or improve the underlying model. They only attempt to influence conversational behavior through existing personalization or instruction features.

## User Responsibility

Users should review AI outputs carefully, especially for decisions involving health, law, finance, employment, safety, or other high-stakes topics.

## Provider Independence

This project is independent and is not affiliated with, endorsed by, or sponsored by OpenAI, Anthropic, Google, or any other AI provider.
