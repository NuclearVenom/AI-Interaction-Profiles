# Roadmap

This roadmap describes possible directions for AI Interaction Profiles. It is intentionally modest: the repository should grow from careful use, feedback, and evaluation rather than product excitement.

## Version 0.1: Repository Foundation

Status: complete

- Defined the initial scope.
- Established the problem, philosophy, and repository foundation.
- Created early documentation, examples, and community files.

Version 0.1 did not include usable Interaction Profiles.

## Version 0.2: Documentation Revision

Status: complete

- Clarified that the project is about interaction behavior, not a task instruction library.
- Strengthened the README and terminology.
- Added the Interaction Manifesto.
- Expanded design principles and examples.
- Replaced intensity-based profile folders with human-centered categories.

Version 0.2 prepared the repository for profile design.

## Version 1.0: Profile Library

Status: current

- Added 14 Interaction Profiles across Communication, Daily, Thinking, and Integrated categories.
- Consolidated each profile into a single `profile.md` page.
- Added Full and Concise versions for copy-paste use.
- Added profile-level asset folders for future screenshots.
- Added a [Profile Index](profiles/index.md) and [Comparison Matrix](docs/comparison-matrix.md).
- Improved navigation and Version 1.0 documentation.

## Future Research Directions

Future work may explore:

- qualitative feedback from real use;
- profile comparison methods;
- evaluation questions for interaction quality;
- failure cases where profile behavior becomes intrusive, weak, or too rigid;
- adaptive interaction systems if the concept proves useful.

These are research directions, not commitments. This repository is not a product plan.

## What Remains Out of Scope

For now, the repository does not include:

- application code;
- browser extension designs;
- APIs;
- model wrappers;
- technical build specifications;
- claims of measured scientific effectiveness.

The current priority is a clear, usable, and coherent interaction profile library.
