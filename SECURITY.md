# Security

AI Interaction Profiles is currently a documentation-only repository. It does not contain executable code, package dependencies, servers, browser extension files, APIs, or automation.

## Reporting Concerns

If you notice a security concern related to this repository, please open a private security advisory if available, or contact the maintainers through the repository's preferred channel.

Examples of relevant concerns may include:

- accidentally published private information;
- malicious content submitted as documentation;
- links that point to unsafe destinations;
- instructions that could encourage unsafe use of AI systems.

## Out of Scope

Because this repository does not implement software, the following are generally out of scope:

- dependency vulnerabilities;
- runtime exploits;
- browser extension permissions;
- API abuse;
- server configuration issues.

If a later version introduces software, this policy should be revised before that work is published.

## Responsible Use

The project encourages careful use of AI assistants. Interaction Profiles should not be designed to bypass safeguards, manipulate users, hide uncertainty, or increase dependency.
