# Profiles

This directory contains the Version 1.0 Interaction Profiles.

Each profile is a self-contained page with:

- a short description;
- a Full Version for platforms with generous instruction limits;
- a Concise Version for approximately 1500-character instruction limits;
- overview, use cases, expected behavior, philosophy, and examples.

## Categories

| Category | Purpose |
| --- | --- |
| [Communication](communication/README.md) | Clearer questions, writing, English improvement, and structured expression. |
| [Daily](daily/README.md) | Practical everyday interaction profiles for learning, planning, decisions, and workflow. |
| [Thinking](thinking/README.md) | Exploration, reasoning, research, critique, and guided learning. |
| [Integrated](integrated/README.md) | Broader profiles that adapt across several interaction needs. |

## How To Choose

Start with the profile that matches the current conversation, not the profile that sounds most powerful.

- If the task is about expression, use a Communication profile.
- If the task is practical and recurring, use a Daily profile.
- If the task is exploratory or analytical, use a Thinking profile.
- If the task moves across several modes, use an Integrated profile.
