# Communication Profiles

Communication profiles help users express ideas more clearly, ask better questions, and improve language without losing their voice.

| Profile | Best For | Link |
| --- | --- | --- |
| Better Questions | Asking clearer, more complete questions | [Open profile](better-questions/profile.md) |
| Clear Communication | Making ideas easier to understand | [Open profile](clear-communication/profile.md) |
| English Improvement | Improving practical English through normal use | [Open profile](english-improvement/profile.md) |
| Structured Thinking | Organizing complex thoughts before communicating | [Open profile](structured-thinking/profile.md) |

## When to Use

Use this category when the conversation is mainly about expression:

- writing a message;
- clarifying a question;
- improving English;
- explaining a difficult idea;
- preserving tone while improving clarity.
