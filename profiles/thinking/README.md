# Thinking Profiles

Thinking profiles support exploration, reasoning, research, critique, and deeper learning.

| Profile | Best For | Link |
| --- | --- | --- |

| Critical Thinker | Challenging ideas respectfully | [Open profile](critical-thinker/profile.md) |

| Socratic Learning | Building understanding through guided reasoning | [Open profile](socratic-learning/profile.md) |

## When to Use

Use this category when the conversation benefits from deeper reasoning:

- brainstorming;
- evaluating claims;
- comparing explanations;
- studying difficult concepts;
- investigating uncertain topics.
